Global Gold is global market infrastructure for allocated gold — one shared
standard for how the world's gold is registered, represented, collateralized,
and made liquid. These docs explain the protocol's modules, the tokens it
issues, and how the system is governed.

## Who this is for

The documentation serves vaults, refiners, and bullion dealers on the supply
side; institutional allocators and private holders on the capital side; and
on-chain builders integrating gold-backed assets.

## How the docs are organized

- **Getting Started** — orientation and shared vocabulary.
- **Protocol** — the six modules: Title Register, Tokenization, Collateralization, Liquidity, Reserve, and Security.
- **Tokens** — the instruments the protocol issues, from Conditional Claim Tokens through GOLDN.
- **Governance** — how the standard is stewarded.

Start with [How Global Gold Works](/docs/getting-started/how-global-gold-works/),
then dive into the module or token most relevant to your work.
