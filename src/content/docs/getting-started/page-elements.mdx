This page exists to demonstrate the docs design system end to end: callouts in
every variant, code blocks with copy, tables, an image, a blockquote, and
nested headings for the "On this page" table of contents.

## Callouts

Four variants, each with a subtle left border and icon.

<Callout type="note">
A **note** carries neutral, supporting context — the stone-bordered default.
</Callout>

<Callout type="tip">
A **tip** highlights a recommended path or a shortcut worth knowing.
</Callout>

<Callout type="warning">
A **warning** flags something that can bite you if ignored.
</Callout>

<Callout type="important">
An **important** callout marks a load-bearing fact you should not skip.
</Callout>

## Typography

Body copy is set in EB Garamond for comfortable reference reading, left-aligned
on a wider measure than the editorial Journal. Inline elements include
**bold**, *italic*, `inline code`, and [internal links](/docs/getting-started/introduction/).

### Lists

Ordered and unordered lists share the brand's gold markers:
